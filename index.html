Data Cleaning in SQL

This project involved cleaning a raw retail sales dataset using SQL in PostgreSQL. The dataset contained inconsistent and unstructured values across key fields such as prices, quantities, transaction dates, and discount indicators. I created duplicate tables to preserve raw data, used regular expressions to clean numeric fields, converted data types, and standardized categories and formats. After cleaning, I removed records with missing critical values to prepare the dataset for analysis.
